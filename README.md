# Actividad 1 - Procesamiento Digital de Señales (EIE_401)

Este repositorio contiene el desarrollo de la primera actividad del curso, enfocada en el análisis de señales, ruido y filtros en frecuencia.

## Resumen
Se trabajó en la generación de señales senoidales, el estudio del impacto del ruido y el procesamiento de imágenes mediante la Transformada de Fourier (FFT), aplicando un filtro de paso bajo circular.

## Herramientas
* Lenguaje: Python
* Librerías: NumPy, Matplotlib y SciPy
* Entorno: Jupyter Notebook en VS Code
